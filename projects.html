Object Classification by Unsupervised Learning with VGG16 & K-Means

Object Classification by Unsupervised Learning with VGG16 & K-Means This project demonstrates an unsupervised approach to object classification by combining deep feature extraction with classical machine learning. Pretrained VGG16 is used to extract high-level visual features from images without using labeled data. These features are then clustered using K-Means to automatically group visually similar objects. The project highlights how transfer learning and unsupervised learning can be integrated to discover hidden patterns in image data, making it suitable for scenarios where labeled datasets are limited or unavailable.
